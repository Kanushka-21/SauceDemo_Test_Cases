1. Application   
   1. SauceDemo is a web-based demo e-commerce application designed to simulate online shopping workflows. It enables testing of key functionalities such as user login, product browsing, cart management, and checkout processes.  
        
2. Key features  
   1. User login  
   2. Product Listing  
   3. Add to Cart  
   4. Remove from cart  
   5. Checkout process  
   6. Logout  
        
3. User Roles  
   1. Standard User  
        
4. User flow  
   1. User logs in \> View products \> Add items to cart/ remove item \> Checkout \> Logout  
        
5. Assumptions  
   1. Network Connection works properly  
   2. User already signed up  
   3. Valid users can login successfully  
   4. Users must login before accessing products

**02.Test Scenarios:**

* Login Functionality  
  * Verify login with valid credentials   
  * Verify login with invalid credentials  
  * Verify login with empty username and password fields  
  * Verify error message is displaying for invalid login  
* Product Listing  
  * Verify products are displayed after successful login  
  * Verify product details visible(name, price, image)  
  * Verify product sorting functionality (if available)  
* Cart Functionality  
  * Verify user can add a product to the cart  
  * Verify user can remove a product from the cart  
  * Verify cart icon updates according to item added/removed  
* Checkout Process  
  * Verify user can proceed to checkout from cart  
  * Verify user can complete checkout with valid details  
  * Verify error when checkout details missing  
* Logout  
  * Verify user can logout successfully
